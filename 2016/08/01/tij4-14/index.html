RTTI

RTTI(Run-Time Type Information) means: At run time, the type of an object is identified.

简单说就是一个类型动态绑定，来确保编程的时候是面向高层接口，运行的时候执行底层派生类的实际动作。就是多态。

可信类

原生类加载器，加载可信类。

类的加载

获取Class的信息

通过Class的静态方法

Class.getName(), Class.getSimpleName(), Class.getCanonicalName()
Class.isInterface()
Class.getInterface()
Class.getSuperClass()
Class.newInstance()
